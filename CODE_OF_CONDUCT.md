# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Our Expectations

- Be respectful and constructive
- Assume good intent and discuss ideas in good faith
- Give actionable feedback instead of personal criticism
- Keep technical disagreement professional

## Unacceptable Behavior

- Harassment, insults, or intimidation
- Discriminatory language or personal attacks
- Repeated low-signal spam or disruptive behavior

## Reporting

If you experience or witness unacceptable behavior, report it privately to `alokgarg003@gmail.com`.

Project maintainers may remove comments, issues, or contributions that do not align with this code of conduct.
